# Feature Plans

Detailed implementation blueprints for upcoming nav-map features. Each plan was generated by analyzing the actual codebase and covers architecture decisions, file maps, data flows, and build sequences.

## Tier 1 — High Impact

| Plan | Description | Key Architecture Decision |
|------|-------------|--------------------------|
| [Time-Travel Slider](./time-travel-slider.md) | Scrub through Git history to see route evolution | Pre-computed snapshot array; CSS transitions on React Flow node styles for enter/exit/morph |
| [Search Preview](./search-preview.md) | Cmd+K shows screenshot thumbnail + neighbors before jumping | Enhanced SearchPanel with preview pane; smooth camera animation via setCenter |
| [AI Explain Flow](./ai-explain-flow.md) | Select nodes → AI describes the flow, finds dead ends | Provider-agnostic LLM adapter (OpenAI/Anthropic/custom); graph serialized as adjacency list |
| [Ghost Trails](./ghost-trails.md) | Walkthrough paths leave fading colored trails | SVG overlay separate from React Flow edges; trails fade over 30s with marching-ant animation |

## Tier 2 — Delightful Polish

| Plan | Description | Key Architecture Decision |
|------|-------------|--------------------------|
| [Living Breathing Graph](./living-breathing-graph.md) | Subtle node pulse + edge particle flow animations | Pure CSS animations (keyframes); breathing rate encodes traffic/recency data |
| [Heat Map Lenses](./heat-map-lenses.md) | Toggle "lenses" to re-color graph by route depth, traffic, staleness | Pluggable lens interface; green-to-red gradient on node backgrounds |
| [Easel Annotation Mode](./easel-annotation-mode.md) | Draw freehand, add sticky notes, circle clusters on the graph | Separate SVG overlay layer above React Flow; annotations stored as JSON |

## Infrastructure

| Plan | Description | Key Architecture Decision |
|------|-------------|--------------------------|
| [CI/CD Auto-Regeneration](./ci-cd-auto-regeneration.md) | Auto-run scanner on deploy, commit updated map | Reusable GitHub Action; diff detection to avoid unnecessary commits |
